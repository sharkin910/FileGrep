FileGrep
========

指定したフォルダ内のテキストファイルから、**指定した文字列に一致する行／一致しない行** を効率よく抽出するための Windows アプリケーションです。

### 主な機能

- **テキスト検索**: 指定した検索文字列に一致する行を一覧表示
- **否定検索**: 「一致しない行」だけを抽出可能
- **複数ファイル対応**: 対象フォルダ配下の複数ファイルを一括検索
- **行単位の結果表示**: ファイル名・行番号・該当行テキストを表示
- **結果のコピー／保存（実装済みであれば）**: 検索結果をクリップボードへコピー、またはファイル出力

※ 実装内容に合わせて、必要に応じて上記機能説明を調整してください。

### 動作環境

- OS: Windows 10 以降
- .NET ランタイム／SDK: 本リポジトリのプロジェクト設定に依存
  - 通常は `.csproj` に記載されている TargetFramework（例: `net6.0-windows` など）に対応した .NET SDK / ランタイムが必要です。

### ビルドと実行方法

1. **リポジトリをクローン**
   - 任意のフォルダで以下を実行（Git を利用する場合）

     ```bash
     git clone <このリポジトリのURL>
     ```

2. **Visual Studio で開く**
   - `FileGrep.sln`（ソリューションファイル）がある場合はそれをダブルクリック
   - または Visual Studio を起動して「ソリューション/プロジェクトを開く」から本フォルダを選択

3. **ビルド**
   - メニューの「ビルド」→「ソリューションのビルド」を実行

4. **実行**
   - メニューの「デバッグ」→「デバッグ開始」または `F5` キーで起動

※ `dotnet` CLI でビルドする場合は、ソリューションまたはプロジェクトのあるディレクトリで `dotnet build` / `dotnet run` を実行してください。

### 使い方（概要）

1. アプリケーションを起動します。
2. 検索対象とするフォルダ（またはファイル）を指定します。
3. 検索したい文字列を入力します。
4. 「一致する行」または「一致しない行」を抽出するモードを選択します（対応している場合）。
5. 検索ボタンを押すと、結果一覧にファイル名・行番号・テキストが表示されます。
6. 必要に応じて結果をコピーしたり、別ファイルに保存します。

実際の UI や操作手順に応じて、ボタン名・メニュー名などは適宜書き換えてください。
